Tema 1: Etiquetas basicas de html, valores y atributos

Html y valores

HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.
Estas etiquetas de marcado son las propias etiquetas HTML. Cuando escribes código en HTML, estás escribiendo etiquetas HTML.Todas las etiquetas HTML están hechas con un número de partes específicas
Las etiquetas mas basicas son:

  • body
  • head
  • div
  • a
  • strong
  • br
  • h1,h2,h3,h4,h5,h6
  • img
  • ol,ul,li
  • p

Atributos

Los atributos de HTML son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo de elemento de HTML.
Un atributo tampoco modifica la funcionalidad por defecto de un tipo de elemento o proporciona funcionalidad a ciertos tipos de elementos incapaces de funcionar correctamente sin ellos.
